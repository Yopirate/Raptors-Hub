# 🦅 Rapaces Maroc - Application Android

[![Android](https://img.shields.io/badge/Android-8.0%2B-green.svg)](https://developer.android.com)
[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.20-blue.svg)](https://kotlinlang.org)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Application Android pour observer les rapaces au Maroc - TOP 20 hotspots avec coordonnées GPS précises.

## 🚀 Installation rapide

### Télécharger l'APK

👉 **[Télécharger la dernière version](../../releases/latest/download/RaptorsMaroc-v0.2.apk)** (8-10 MB)

### Installer sur votre téléphone

1. Télécharger le fichier APK
2. Ouvrir le fichier sur votre téléphone Android
3. Autoriser "Installation depuis sources inconnues" si demandé
4. Installer l'application
5. Profiter ! 🦅

## ✨ Fonctionnalités

- ✅ **TOP 20 hotspots** d'observation des rapaces au Maroc
- ✅ **Coordonnées GPS précises** pour chaque site
- ✅ **Mode hors-ligne** - Toutes les données embarquées
- ✅ **Liens directs** vers Google Maps, eBird, BirdLife
- ✅ **Filtres intelligents** par importance, région, espèces
- ✅ **Interface Material Design 3** moderne et fluide
- ✅ **Support multilingue** FR/AR/EN

## 🦅 TOP 5 Hotspots

1. **Jbel Moussa** (35.898°N, -5.411°W) - 12 espèces - >30,000 rapaces/automne
2. **Cap Spartel** (35.793°N, -5.920°W) - 15 espèces - >250,000 cigognes
3. **Parc National Souss-Massa** (30.075°N, -9.639°W) - 18 espèces - Ibis chauve
4. **Vallée du Souss** (30.365°N, -9.584°W) - 14 espèces - Aigles hivernants
5. **Zaida** (32.795°N, -4.928°W) - 10 espèces - Aigle royal

## 🛠️ Compiler depuis les sources

### Prérequis

- Android Studio Hedgehog 2023.1.1+
- JDK 17
- Android SDK 34

### Compilation

```bash
# Cloner le dépôt
git clone https://github.com/VOTRE-USERNAME/raptors-hub.git
cd raptors-hub

# Compiler APK debug
./gradlew assembleDebug

# APK généré dans : app/build/outputs/apk/debug/app-debug.apk
```
